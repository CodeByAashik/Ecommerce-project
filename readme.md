# 1. ShopCenter 

### A simple e-commerce frontend built with React and modern UI components. The project includes product listing, product detail view, cart functionality, and a smooth user experience focused layout.

# 2. Features
- Product listing grid
- Product details page 
- Add/Remove items from cart
- Resposive layout
- Search product by name
- Place order
- Track order

# 3. Tech Stack
| Layer | Tools | 
|-------|-------|
|Language|Javascript|
|Framweork|React|
|Styling|CSS|

# 4. Installation
## 4.1 Clone repo
remote add origin https://github.com/CodeByAashik/Ecommerce-project.git

## 4.2 Go into folder
cd project

## 4.3 Install deps
npm install

## 4.4 Run dev server
npm run dev
